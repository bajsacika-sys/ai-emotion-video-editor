# ai-emotion-video-editor
AI Emotion Video Editor — MVP-проект, который анализирует эмоции на лице в видео и автоматически применяет визуальные эффекты, фильтры и монтаж под настроение человека. Проект создан специально для хакатона 2025. Простой в запуске: поддержка Google Colab, Gradio и автоматической обработки видео.
